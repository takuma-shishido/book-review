## この本で触れられている物 (細かなものはキリがないため省略)
- アセンブラ (nask)
- C言語

## 良かった点
- 内容が充実している (およそ700ページ） 
- コードの後に作者の雑談のような部分がありそこで説明がされていて楽しく学ぶことができる
- OSの基礎部分(カーネルなど)から自作していてかなり広い部分の知識を得られる

## 悪かった点
- Amazonレビューにもあるがページが多く読むのに時間がかかる
- 理解するためにある程度の知識が必要になる
